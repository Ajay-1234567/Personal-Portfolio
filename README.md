- Next.js 16
- React 19
- Tailwind CSS 4

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser for local development.

**Deploy for Free:** Your portfolio will be live at `ajay-kumar-portfolio.vercel.app` (or similar)
